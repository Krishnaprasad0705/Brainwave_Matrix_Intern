# 🚗 BMW Sales Dashboard (India – 2024–2025)

An interactive Power BI dashboard analyzing BMW’s revenue, profit, and unit sales in the Indian market.

---

## 🔍 Key Features

- Total Revenue, Profit, and Units Sold
- Monthly Revenue vs. Profit trend
- Model-wise Profit Performance (Top 5)
- Region-wise Revenue and Profit Margin
- Interactive filters and clean UI with BMW branding

---

## 🛠 Tools & Technologies Used

- Power BI Desktop
- DAX
- Microsoft Excel
